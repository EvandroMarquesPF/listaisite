Termos de Uso

ListAI - Lista de Compras Inteligente com IA

Última atualização: 26 de setembro de 2025

1. Aceitação dos Termos

Ao utilizar o aplicativo ListAI, você concorda em cumprir e estar vinculado a estes Termos de Uso. Se você não concordar com qualquer parte destes termos, não deve usar nosso aplicativo.

2. Descrição do Serviço

O ListAI é um aplicativo móvel que oferece:

  • Lista de compras inteligente com Inteligência Artificial
  • Scanner de código de barras para reconhecimento de produtos
  • Reconhecimento de texto por câmera
  • Integração com Google Maps para localização
  • Sincronização em nuvem com Firebase
  • Autenticação com Google
  • Relatórios avançados de gastos
  • Sistema de compartilhamento inteligente

3. Uso do Aplicativo

3.1 Uso Permitido

Você pode usar o ListAI para:

  • Criar e gerenciar listas de compras pessoais
  • Escanear códigos de barras de produtos
  • Utilizar recursos de IA para sugestões
  • Sincronizar dados entre seus dispositivos
  • Compartilhar listas com familiares e amigos

3.2 Uso Proibido

É proibido usar o aplicativo para:

  • Atividades ilegais ou não autorizadas
  • Violar direitos de propriedade intelectual
  • Interferir no funcionamento do aplicativo
  • Coletar dados de outros usuários
  • Usar o aplicativo para fins comerciais não autorizados

4. Conta de Usuário

4.1 Autenticação

O ListAI utiliza autenticação com Google para:

  • Proteger informações pessoais

4.2 Responsabilidades do Usuário

Você é responsável por:
